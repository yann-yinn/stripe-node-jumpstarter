# node-stripe-starter-auth

## getting started

1. Créer une base de données MongoDB avec une table `users`

2. intaller le code:

```sh
cd client
npm install
# éditez les variables d'environnements!
cp .env.example .env.local
# démarrer l'API
npm run start

cd -
cd server
npm install
# éditez les variables d'environnements!
cp .env.example .env
# démarrer l'API
npm run serve
```
